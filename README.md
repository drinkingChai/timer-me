# When Will

Timer app to count down.

#### Features

- Displays
  - Months
  - Weeks
  - Days
  - Hours
  - Minutes
  - Seconds
- Options
  - Include Weekdays
  - Include Weekends

#### Future

- Holidays from [holidayapi.com](http://holidayapi.com)
- Email alerts
