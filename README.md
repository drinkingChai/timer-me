# When Will

Timer app to count down.

#### Features

- Holidays from [holidayapi.com](http://holidayapi.com)
- Choose what to show from:
- Displays
  - Weeks
  - Days
  - Hours
  - Minutes
  - Seconds
- Options
  - Include Weekdays
  - Include Weekends

#### Future
